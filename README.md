# Actividad 2 - Flujo de Control Parte 2

## Descripción

Este proyecto fue desarrollado en C# utilizando estructuras de control repetitivas (for) y condicionales (if-else).

El programa solicita la cantidad de estudiantes, permite ingresar el nombre y las cuatro calificaciones de cada uno, calcula el promedio y determina si el estudiante aprobó o reprobó. Finalmente, muestra toda la información organizada en forma de tabla mediante tabulación.

## Funcionalidades

- Solicita la cantidad de estudiantes.

- Permite ingresar el nombre de cada estudiante.

- Solicita las cuatro calificaciones.

- Calcula el promedio de las calificaciones.

- Determina si el estudiante está aprobado o reprobado.

- Muestra los resultados organizados en una tabla.

## Tecnologías utilizadas

- Lenguaje: C#

- IDE: Visual Studio

- Consola de Windows
